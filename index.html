Mutimodal Large Language Models

WebUIBench: A Comprehensive Benchmark for Evaluating Multimodal Large Language Models in WebUI-to-Code

We propose WebUIBench, a benchmark that systematically evaluates MLLMs’ web development abilities across four key sub-capabilities using real-world data and multi-view metrics.

Code Generation Safety

LLMs Caught in the Crossfire: Malware Requests and Jailbreak Challenges

This work introduces the MalwareBench benchmark dataset, which innovatively combines the issue of malware generation and LLM attack methods. It highlights the significant challenges different types of LLMs currently face under malicious code instructions, providing direction for future research.

2025-05-15
